# Number Facts SPA

A simple React + Vite single-page application to display interesting facts about numbers using the [numbersapi.com](http://numbersapi.com) API.

## Features
- Choose category: Math, Trivia, or Date
- Enter a number or get a random fact
- Input validation and error handling
- Clean UI with TailwindCSS
- Routing with React Router

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
